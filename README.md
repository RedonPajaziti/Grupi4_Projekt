Time Management Program (C++)

Project Description

This is a group project written in C++ that simulates a simple digital clock.
The program allows the user to enter a time (hours, minutes, and seconds), automatically validates the input, adds one second to the given time, and then displays the updated time.
The project demonstrates the use of:
  struct in C++
  Member functions
  Input validation
  Basic time logic (seconds, minutes, hours)
